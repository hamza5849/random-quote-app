# Random Quote App

A simple React app that shows random quotes. Built with React, Vite, and Tailwind CSS.

## Features
- Display random quotes on button click
- Responsive design using Tailwind CSS
- Fast and lightweight

## Prerequisites
- Node.js installed
- npm installed
